## Solutions

### Part A Solutions
1. $i^1 = i$
2. $i^2 = -1$
3. $i^3 = -i$
4. $i^4 = 1$
5. $i^5 = i$ (cycle repeats)
6. $i^{10} = i^2 = -1$ (10 ÷ 4 = 2 remainder 2)
7. $i^{15} = i^3 = -i$ (15 ÷ 4 = 3 remainder 3)
8. $i^{20} = i^4 = 1$ (20 ÷ 4 = 5 remainder 0)
9. $i^{27} = i^3 = -i$ (27 ÷ 4 = 6 remainder 3)
10. $i^{100} = i^4 = 1$ (100 ÷ 4 = 25 remainder 0)

### Part B Solutions
11. Real: 5, Imaginary: 3
12. Real: -2, Imaginary: 7
13. Real: 4, Imaginary: -6
14. Real: 0, Imaginary: 8
15. Real: -10, Imaginary: 0
16. Real: 0, Imaginary: 9

### Part C Solutions
17. $\sqrt{-16} = 4i$
18. $\sqrt{-25} = 5i$
19. $\sqrt{-1} = i$
20. $\sqrt{-49} = 7i$
21. $\sqrt{-100} = 10i$
22. $\sqrt{-8} = i\sqrt{8} = i \cdot 2\sqrt{2} = 2\sqrt{2}i$

### Part D Solutions
23. $(3 + 4i) + (2 + 5i) = 5 + 9i$
24. $(7 - 2i) + (-3 + 6i) = 4 + 4i$
25. $(5 + i) + (5 - i) = 10 + 0i = 10$ (conjugates sum to real!)
26. $(-1 + 8i) + (4 - 3i) = 3 + 5i$
27. $(6 + 2i) + 3i = 6 + 5i$

### Part E Solutions
28. $(8 + 5i) - (3 + 2i) = 5 + 3i$
29. $(4 - 7i) - (1 - 3i) = 3 - 4i$
30. $(10 + 6i) - (10 - 6i) = 0 + 12i = 12i$
31. $(2 + i) - (5 + 4i) = -3 - 3i$
32. $(9 - 2i) - 4i = 9 - 6i$

### Part F Solutions
33. Conjugate of $3 + 4i$ is $3 - 4i$
34. Conjugate of $5 - 2i$ is $5 + 2i$
35. Conjugate of $-6 + 7i$ is $-6 - 7i$
36. Conjugate of $8i$ is $-8i$
37. Conjugate of $-3i$ is $3i$
38. Conjugate of $10$ is $10$

### Part G Solutions
39. $|3 + 4i| = \sqrt{9 + 16} = \sqrt{25} = 5$
40. $|5 + 12i| = \sqrt{25 + 144} = \sqrt{169} = 13$
41. $|8 - 6i| = \sqrt{64 + 36} = \sqrt{100} = 10$
42. $|-7 + 24i| = \sqrt{49 + 576} = \sqrt{625} = 25$
43. $|6i| = \sqrt{0 + 36} = 6$
44. $|15| = \sqrt{225 + 0} = 15$

### Part H Solutions
45. $x = 5$, $y = 3$
46. $2x = 6$ → $x = 3$; $4 = y + 1$ → $y = 3$
47. $x - 2 = 5$ → $x = 7$; $y + 3 = 7$ → $y = 4$

### Part I Solutions
48. $(2 + 3i) + (4 - i) - (1 + 2i) = (2 + 4 - 1) + (3 - 1 - 2)i = 5 + 0i = 5$
49. $(5 + 2i) - (3 - 4i) + (-1 + i) = (5 - 3 - 1) + (2 + 4 + 1)i = 1 + 7i$
50. $(6 - i) + (2 + 5i) - (3 + 3i) = (6 + 2 - 3) + (-1 + 5 - 3)i = 5 + i$

### Part J Solutions
51. $(3 + 4i) + (3 - 4i) = 6$ (always real!)
52. $(5 - 2i) - (5 + 2i) = -4i$ (always purely imaginary!)
53. $i + i^2 + i^3 + i^4 = i + (-1) + (-i) + 1 = 0$
54. $|z|^2 = |2 + 3i|^2 = (\sqrt{13})^2 = 13$
    $z \times \overline{z} = (2 + 3i)(2 - 3i) = 4 + 9 = 13$
    **They're equal!** This is always true: $|z|^2 = z \times \overline{z}$
