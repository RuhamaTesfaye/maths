## ğŸ”‘ Key Concepts Summary

### Regular Numbers
- **Multiplication:** Repeated addition; same signs â†’ positive, different signs â†’ negative
- **Division:** Inverse of multiplication; "Keep, Change, Flip" for fractions

### Complex Numbers
- **Form:** $z = a + bi$ where $i^2 = -1$
- **Multiplication:** Use FOIL, replace $i^2$ with $-1$
- **Division:** Multiply by conjugate of denominator
