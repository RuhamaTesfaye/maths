## Solutions

### Part A Solutions
1. $\frac{2 + i}{1 - i} \times \frac{1 + i}{1 + i} = \frac{2 + 2i + i + i^2}{2} = \frac{1 + 3i}{2} = \frac{1}{2} + \frac{3}{2}i$

2. $\frac{3 + 2i}{1 + i} \times \frac{1 - i}{1 - i} = \frac{3 - 3i + 2i - 2i^2}{2} = \frac{5 - i}{2} = \frac{5}{2} - \frac{1}{2}i$

3. $\frac{4 + 3i}{2 + i} \times \frac{2 - i}{2 - i} = \frac{8 - 4i + 6i - 3i^2}{5} = \frac{11 + 2i}{5} = \frac{11}{5} + \frac{2}{5}i$

4. $\frac{5 - i}{3 + i} \times \frac{3 - i}{3 - i} = \frac{15 - 5i - 3i + i^2}{10} = \frac{14 - 8i}{10} = \frac{7}{5} - \frac{4}{5}i$

5. $\frac{1 + 4i}{2 - i} \times \frac{2 + i}{2 + i} = \frac{2 + i + 8i + 4i^2}{5} = \frac{-2 + 9i}{5} = -\frac{2}{5} + \frac{9}{5}i$

### Part B Solutions
6. $\frac{3 - 2i}{1 - 2i} \times \frac{1 + 2i}{1 + 2i} = \frac{3 + 6i - 2i - 4i^2}{5} = \frac{7 + 4i}{5} = \frac{7}{5} + \frac{4}{5}i$

7. $\frac{-4 + 3i}{2 + i} \times \frac{2 - i}{2 - i} = \frac{-8 + 4i + 6i - 3i^2}{5} = \frac{-5 + 10i}{5} = -1 + 2i$

8. $\frac{5 + 2i}{-1 + 3i} \times \frac{-1 - 3i}{-1 - 3i} = \frac{-5 - 15i - 2i - 6i^2}{10} = \frac{1 - 17i}{10} = \frac{1}{10} - \frac{17}{10}i$

9. $\frac{-6 - i}{3 - 2i} \times \frac{3 + 2i}{3 + 2i} = \frac{-18 - 12i - 3i - 2i^2}{13} = \frac{-16 - 15i}{13} = -\frac{16}{13} - \frac{15}{13}i$

10. $\frac{2 - 5i}{-2 + i} \times \frac{-2 - i}{-2 - i} = \frac{-4 - 2i + 10i + 5i^2}{5} = \frac{-9 + 8i}{5} = -\frac{9}{5} + \frac{8}{5}i$

### Part C Solutions
11. $\frac{6 + 9i}{3} = 2 + 3i$
12. $\frac{10 - 15i}{5} = 2 - 3i$
13. $\frac{8 + 12i}{4} = 2 + 3i$
14. $\frac{-14 + 21i}{7} = -2 + 3i$

### Part D Solutions
15. $\frac{4 + 6i}{2i} \times \frac{-i}{-i} = \frac{-4i - 6i^2}{2} = \frac{6 - 4i}{2} = 3 - 2i$

16. $\frac{3 - 5i}{i} \times \frac{-i}{-i} = \frac{-3i + 5i^2}{1} = -5 - 3i$

17. $\frac{10 + 4i}{3i} \times \frac{-i}{-i} = \frac{-10i - 4i^2}{3} = \frac{4 - 10i}{3} = \frac{4}{3} - \frac{10}{3}i$

18. $\frac{2 - 8i}{-2i} \times \frac{i}{i} = \frac{2i - 8i^2}{2} = \frac{8 + 2i}{2} = 4 + i$

### Part E Solutions
19. $\frac{3 + 4i}{i} = 4 - 3i$ (imaginary becomes real with sign flip)
20. $\frac{5 - 2i}{i} = -2 - 5i$
21. $\frac{-1 + 7i}{i} = 7 + i$
22. $\frac{8 - 3i}{i} = -3 - 8i$

### Part F Solutions
23. $\frac{1}{2 + i} = \frac{2 - i}{5} = \frac{2}{5} - \frac{1}{5}i$

24. $\frac{1}{3 - 4i} = \frac{3 + 4i}{25} = \frac{3}{25} + \frac{4}{25}i$

25. $\frac{1}{1 + i} = \frac{1 - i}{2} = \frac{1}{2} - \frac{1}{2}i$

26. $\frac{1}{5i} = \frac{-i}{5} = -\frac{1}{5}i$

### Part G Solutions
27. $(1 + i)(2 - i) = 3 + i$, so $\frac{3 + i}{1 + 2i} \times \frac{1 - 2i}{1 - 2i} = \frac{3 - 6i + i - 2i^2}{5} = \frac{5 - 5i}{5} = 1 - i$

28. $(3 + 2i)(1 - i) = 5 - i$, so $\frac{5 - i}{2 + i} \times \frac{2 - i}{2 - i} = \frac{10 - 5i - 2i + i^2}{5} = \frac{9 - 7i}{5} = \frac{9}{5} - \frac{7}{5}i$

29. $(2 + i)^2 = 3 + 4i$, so $\frac{3 + 4i}{3 - i} \times \frac{3 + i}{3 + i} = \frac{9 + 3i + 12i + 4i^2}{10} = \frac{5 + 15i}{10} = \frac{1}{2} + \frac{3}{2}i$

30. $(4 - 3i)(1 + i) = 7 + i$, so $\frac{7 + i}{2 - 3i} \times \frac{2 + 3i}{2 + 3i} = \frac{14 + 21i + 2i + 3i^2}{13} = \frac{11 + 23i}{13} = \frac{11}{13} + \frac{23}{13}i$

### Part H Solutions

32. $\frac{2}{1 + i} = 1 - i$, $\frac{2}{1 - i} = 1 + i$, so $(1 - i) - (1 + i) = -2i$

33. $\frac{3 + i}{2} + \frac{1 - 2i}{2} = \frac{4 - i}{2} = 2 - \frac{1}{2}i$

### Part I Solutions

### Part K Solutions
39. $\frac{1}{1+i} + \frac{1}{1-i} + \frac{1}{i} = \frac{1-i}{2} + \frac{1+i}{2} - i = 1 - i$

40. $(1+i)^2 = 2i$, $(1-i)^2 = -2i$, so $\frac{2i}{-2i} = -1$

41. First: $\frac{2+3i}{1-i} = \frac{-1+5i}{2}$, second: $\frac{1+i}{2-i} = \frac{3+3i}{5}$
    Product: $\frac{(-1+5i)(3+3i)}{10} = \frac{-18+12i}{10} = -\frac{9}{5} + \frac{6}{5}i$

42. Let $z = a + bi$. Then $(a+bi)^2 = a^2 - b^2 + 2abi = i$
    So $a^2 - b^2 = 0$ and $2ab = 1$
    From first: $a = \pm b$. If $a = b$: $2a^2 = 1$, so $a = \pm\frac{1}{\sqrt{2}}$
    **Answer:** $z = \frac{1}{\sqrt{2}} + \frac{1}{\sqrt{2}}i$ or $z = -\frac{1}{\sqrt{2}} - \frac{1}{\sqrt{2}}i$

### Part L Solutions
43. $(3+2i)(1-i) = 5 - i$, so $\frac{5-i}{2+i} = \frac{9-7i}{5} = \frac{9}{5} - \frac{7}{5}i$

44. $(2-i)^2 = 3-4i$, $(2+i)^2 = 3+4i$, sum = $6$, so $\frac{6}{4} = \frac{3}{2}$

45. $|3+4i|^2 = 25$, so $\frac{25}{3-4i} = \frac{25(3+4i)}{25} = 3 + 4i$

46. $i^{17} = i$, $i^{18} = -1$, $i^{19} = -i$, $i^{20} = 1$
    $\frac{i - 1}{-i + 1} = \frac{i-1}{1-i} \times \frac{1+i}{1+i} = \frac{i + i^2 - 1 - i}{2} = \frac{-2}{2} = -1$
